<h1>ALX SE Programme Project 0x04. Loops, conditions and parsing</h1>


This project encompasses solutions to various tasks centered around loops, conditions, and parsing as part of the ALX SE Programme. Each task is approached with dedicated scripts that emphasize mastering essential techniques in shell scripting.

---

<h1>Content</h1>

| Task | Files | Description |
| ----- | ----- | ------ |
| 0 | [0-RSA_public_key.pub](./0-RSA_public_key.pub) | Contains SSH Public key. |
| 1 | [1-for_best_school](./1-for_best_school) | Script that displays Best School 10 times using for loop. |
| 2 | [2-while_best_school](./2-while_best_school) | Script that displays Best School 10 times using while loop. |
| 3 | [3-until_best_school](./3-until_best_school) | Script that displays Best School 10 times using until loop. |
| 4 | [4-if_9_say_hi](./4-if_9_say_hi) | Script that displays Best School 10 times, but for the 9th iteration, displays Best School and then Hi on a new line. |
| 5 | [5-4_bad_luck_8_is_your_chance](./5-4_bad_luck_8_is_your_chance) | Script that loops from 1 to 10 and displays bad luck for the 4th loop iteration, displays good luck for the 8th loop iteration and displays Best School for the other iterations. |
| 6 | [6-superstitious_numbers](./6-superstitious_numbers) | Script that displays numbers from 1 to 20 and displays 4 and then bad luck from China for the 4th loop iteration, displays 9 and then bad luck from Japan for the 9th loop iteration and displays 17 and then bad luck from Italy for the 17th loop iteration |
| 7 | [7-clock](./7-clock) | Script that displays the time for 12 hours and 59 minutes. |
| 8 | [8-for_ls](./8-for_ls) | Script that displays The content of the current directory in a list format where only the part of the name after the first dash is displayed. |
| 9 | [9-to_file_or_not_to_file](./9-to_file_or_not_to_file) | Script that gives you information about the school file. |
| 10 | [10-fizzbuzz](./10-fizzbuzz) | FizzBuzz script. |
| 11 | [100-read_and_cut](./100-read_and_cut) | Script that displays the content of the file /etc/passwd, only displays username, user id and home directory path for the user. |
| 12 | [101-tell_the_story_of_passwd](./101-tell_the_story_of_passwd) | Script that displays the content of the file /etc/passwd, using the while loop + IFS. |
| 13 | [102-lets_parse_apache_logs](./102-lets_parse_apache_logs) | Script that displays the visitor IP along with the HTTP status code from the Apache log file. |
| 14 | [103-dig_the-data](./103-dig_the-data) | cript that groups visitors by IP and HTTP status code, and displays this data. |
