<h1>ALX SE Programme Project 0x05. Processes and signals</h1>

This project consists of a series of tasks focused on processes and signals within the ALX SE Programme. Each task is tackled with dedicated scripts aimed at mastering fundamental techniques in managing processes, handling signals, and optimizing system performance through efficient process management. Participants will engage in practical exercises designed to deepen their understanding and proficiency in these critical aspects of shell scripting, ensuring they are well-prepared to apply these skills in real-world scenarios.

---

<h1>Content</h1>

| Task | Files | Description |
| ----- | ----- | ------ |
| 0 | [0-what-is-my-pid](./0-what-is-my-pid) | Script that displays its own PID. |
| 1 | [1-list_your_processes](./1-list_your_processes) | Script that displays a list of currently running processes. |
| 2 | [2-show_your_bash_pid](./2-show_your_bash_pid) | Script that displays lines containing the bash word, thus allowing you to easily get the PID of your Bash process. |
| 3 | [3-show_your_bash_pid_made_easy](./3-show_your_bash_pid_made_easy) | Script that displays the PID, along with the process name, of processes whose name contain the word bash. |
| 4 | [4-to_infinity_and_beyond](./4-to_infinity_and_beyond) | Script that displays To infinity and beyond indefinitely. |
| 5 | [5-dont_stop_me_now](./5-dont_stop_me_now) | Script that stops 4-to_infinity_and_beyond process using kill. |
| 6 | [6-stop_me_if_you_can](./6-stop_me_if_you_can) | Script that stops 4-to_infinity_and_beyond process using pkill. |
| 7 | [7-highlander](./7-highlander) | Script that displays to infinity and beyond indefinitely with a sleep 2 in between each iteration, I am invincible!!! when receiving a SIGTERM signal. |
| 8 | [8-beheaded_process](./8-beheaded_process) | Script that kills the process 7-highlander. |
| 9 | [100-process_and_pid_file](./100-process_and_pid_file) | Script that creates the file /var/run/myscript.pid containing its PID, displays To infinity and beyond indefinitely, displays I hate the kill command when receiving a SIGTERM signal, displays Y U no love me?! when receiving a SIGINT signal and deletes the file /var/run/myscript.pid and terminates itself when receiving a SIGQUIT or SIGTERM signal |
| 10 | [manage_my_process](./manage_my_process) | Script that indefinitely writes I am alive! to the file /tmp/my_process, in between every I am alive! message, the program should pause for 2 seconds. |
| 10 | [101-manage_my_process](./101-manage_my_process) | Script  that manages manage_my_process. |
| 11 | [102-zombie.c](./102-zombie.c) | C Program that creates 5 zombie processes. |
