# 0x05. Processes and signals
This project (0x05) is part of the [alx-system_engineering-devops](https://github.com/AbdullahHR10/alx-system_engineering-devops) repository, It focuses on learning processes and signals and includes various tasks to utilize the information learned

## Learning Objectives
### General
- What is a PID
- What is a process
- How to find a process’ PID
- How to kill a process
- What is a signal
- What are the 2 signals that cannot be ignored
