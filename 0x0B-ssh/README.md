# ALX SE Programme Project 0x0B. SSH

This project (0x0B) is part of the [alx-system_engineering-devops](https://github.com/AbdullahHR10/alx-system_engineering-devops) repository, It focuses on learning SSH and includes various tasks to utilize the information learned

## Learning Objectives
### General
- What is a server
- Where servers usually live
- What is SSH
- How to create an SSH RSA key pair
- How to connect to a remote host using an SSH RSA key pair
- The advantage of using #!/usr/bin/env bash instead of /bin/bash

---

# Content

| Task | Files | Description |
| ----- | ----- | ------ |
| 0. Use a private key | [0-use_a_private_key](./0-use_a_private_key) | Script that uses ssh to connect to your server using the private key ~/.ssh/school with the user ubuntu. |
| 1. Create an SSH key pair | [1-create_ssh_key_pair](./1-create_ssh_key_pair) | Script that creates an RSA key pair. |
| 2. Client configuration file | [2-ssh_config](./2-ssh_config) | Configures an SSH configuration file for the local SSH client. |
| 3. Let me in! | None | Added the SSH public key provided by the programme to the server so that it can connect using the ubuntu user. |
| 4. Client configuration file (w/ Puppet) | [100-puppet_ssh_config.pp](./100-puppet_ssh_config.pp) | Configures an SSH configuration file for the local SSH client using Puppet. |
