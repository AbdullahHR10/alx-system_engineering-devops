# ALX SE Programme Project 0x0A. Configuration management

This project (0x09) is part of the alx-system_engineering-devops repository, It focuses on learning configuration management and includes various tasks to utilize the information learned

## How to install Puppet
```
apt-get install -y ruby=1:2.7+1 --allow-downgrades
apt-get install -y ruby-augeas
apt-get install -y ruby-shadow
apt-get install -y puppet
```
## How to install puppet-lint
```
gem install puppet-lint
```
