# 0x08. Networking basics #1

This project (0x08) is part of the alx-system_engineering-devops repository, It focuses on learning networking basics and includes various tasks to utilize the information learned

## Learning Objectives
### General
- What is localhost/127.0.0.1
- What is 0.0.0.0
- What is /etc/hosts
- How to display your machine’s active network interfaces

---

# Content 

| Task | Files | Description |
| ----- | ----- | ------ |
| 0. Change your home IP | [0-change_your_home_IP](./0-change_your_home_IP) | Script that configures an Ubuntu server with the requirements: localhost resolves to 127.0.0.2 and facebook.com resolves to 8.8.8.8, The checker is running on docker. | 
| 1. Show attached IPs | [1-show_attached_IPs](./1-show_attached_IPs) | Script that displays all active IPv4 IPs on the machine it’s executed on. |
| 2. Port listening on localhost | [100-port_listening_on_localhost](./100-port_listening_on_localhost) | Script that listens on port 98 on localhost. |
